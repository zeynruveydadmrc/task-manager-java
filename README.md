# 🚀 Java Task Manager Application

This project is a console-based task management tool designed to demonstrate fundamental **Object-Oriented Programming (OOP)** principles in Java.

## 🛠 Features
- **Task Management:** Create and list tasks dynamically.
- **Status Tracking:** Mark tasks as completed with real-time status updates.
- **OOP Architecture:** Clean separation of concerns using `Task`, `TaskManager`, and `Main` classes.

## 💻 Tech Stack & Concepts
- **Language:** Java (JDK 23)
- **Data Structures:** `ArrayList` for dynamic data handling.
- **Concepts:** Encapsulation, Constructors, and Conditional Logic.
- **Version Control:** Git/GitHub.

## 📖 How to Run
1. Clone the repository.
2. Compile: `javac *.java`
3. Run: `java Main`
